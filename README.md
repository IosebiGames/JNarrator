# JNarrator

JNarrator is Desktop GUI Application Made in Java which represents a Simple Narrator. it can read aloud any text filled by User right after "Narrate" Button is being pressed.

## Features
* **Text Narration:** Reads any text entered manually into the application window.
* **File Support:** Allows you to upload and narrate text directly from .txt files.
* **Format Validation:** Only accepts files that strictly end with the .txt extension.

## Frameworks & Libraries Used
* **Java Swing & AWT:** Swing is framework from JRE System User Library used for Desktop GUIs and their components, it is part of Java Fundation Classes (JFC) and is often paired with Abstract Window Toolkit for additional functions.
* **FlatLaf:** A small, look-and-feel library used to provide a clean and comfortable design.
* **FreeTTS:** The core speech synthesis library used for narrating the text.

## How to Run
1. Download the jar file from this Repository's Latest Releases.
2. Double-click the file to run it.
3. Alternatively, right-click the file and select "Open".

## System Requirements

To run this application, You must have the Latest JDK (Java Development Kit) Installed and Java Home set from your Environment Variables.
It is optional, but still good to consider Installing Java itself too, for better control and Basic JRE.
